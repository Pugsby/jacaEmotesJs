Usage:

Just use "_EMOTENAME_" in your text and parse it with the following code:

parseJacaEmotes(document.body.innerHTML) .then(result => document.body.innerHTML = result);

or, if you want a simpler approach, use class "parse" on the elements you want to parse:

var toParse = document.getElementsByClassName("parse"); Array.from(toParse).forEach(element => { parseJacaEmotes(element.innerHTML) .then(result => element.innerHTML = result); });

Including the script:

You have two (easy) options.

A. Download the minified script and include it in your HTML file.

<script src="jacaEmotes.minified.js"></script>

B. Get it from this website instead (less reliable):

<script src="https://pugsbyy.github.io/jacaEmotesJs/jacaEmotes.minified.js"></script>

Or, you can use the hardest approach which does nothing better.

You have to include static.png in EVERY SINGLE FOLDER where you're using this script.

And obviously include the script in your HTML file.
